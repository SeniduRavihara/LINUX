# Linux Performance Engineering & Documentation

This repository documents my engineering approach to optimizing a 2-year-old i5 8th gen, 16GB RAM laptop running Fedora Linux. Through deep kernel-level tuning, memory management, and proactive stability engineering, I have transformed this workstation into a high-performance multitasking machine capable of running heavy development workloads (Android Studio, multiple IDEs, numerous Chrome tabs, and background build processes).
